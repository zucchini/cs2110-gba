CS2110 GBA

Welcome!

This website is a showcase of GameBoy Advance games developed by students as an assignment for Georgia Tech's CS 2110 starting from Fall 2017. A selection of games is available for playing below.

The default key mappings are the same as VBA. They are as follows (in GBA -> Keyboard pairs): A -> Z, B -> X, L -> A, R -> S, SELECT -> Backspace, START -> Enter, D-Pad: Arrow Keys. You can modify the mappings using the settings menu (click on the key in the list, then tap the key you want to map it to).
